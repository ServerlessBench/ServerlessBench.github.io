版本速览

Dubbo 社区目前主力维护的有 2.6.x 和 2.7.x 两大版本，其中，

  • 2.6.x 主要以 bugfix 和少量 enhancements 为主，因此能完全保证稳定性
  • 2.7.x 作为社区的主要开发版本，得到持续更新并增加了大量新 feature 和优化，同时也带来了一些稳定性挑战
